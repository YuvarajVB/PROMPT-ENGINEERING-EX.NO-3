# Aim

To write prompts for different Prompt Engineering techniques and evaluate their outputs using various prompting methods such as Straightforward Prompting, Tabular Format Prompting, Preceding Question Prompting, and Missing Word Prompting.

---

# Introduction

Prompt Engineering is an important technique in Generative Artificial Intelligence that involves designing effective instructions to guide AI systems toward generating meaningful and accurate outputs. Different prompting styles influence the clarity, reasoning ability, and response quality of Large Language Models (LLMs).

This experiment focuses on studying four major prompt types:

1. Straightforward Prompts
2. Tabular Format Prompting
3. Preceding Question Prompting
4. Missing Word Prompting

The report evaluates how these prompting techniques improve communication with AI systems through practical examples and scenario-based testing.

---

# Objective

1. To understand different prompt engineering techniques.
2. To design prompts using multiple prompting styles.
3. To evaluate the effectiveness of prompts.
4. To analyze AI-generated responses.
5. To study how prompt structure affects output quality.

# Use Case Chosen

## Smart Healthcare Assistant System

The selected use case is a Smart Healthcare Assistant that helps users understand medical concepts, healthcare technologies, vaccination processes, and treatment-related information using AI-powered conversational systems.

# Prompt Types and Examples

---

# 1. Straightforward Prompts

## Definition

Straightforward prompts are simple, direct, and clear instructions given to the AI model without additional context or formatting requirements.

## Examples

### Prompt 1:

“Define telemedicine in one sentence.”

### AI Response:

“Telemedicine is the use of digital communication technologies to provide remote healthcare services and medical consultations.”

---

### Prompt 2:

“List three advantages of electronic health records.”

### AI Response:

1. Improved patient data accessibility
2. Faster medical documentation
3. Better coordination among healthcare providers

---

## Evaluation

| Criteria             | Observation |
| -------------------- | ----------- |
| Clarity              | High        |
| Response Speed       | Fast        |
| Depth of Information | Moderate    |
| Accuracy             | Good        |

# 2. Tabular Format Prompting

## Definition

Tabular prompting requests the AI to organize information into rows and columns for easier comparison and readability.

## Examples

### Prompt 1:

“Compare traditional healthcare and telemedicine in a table.”

### AI Response:

| Feature           | Traditional Healthcare | Telemedicine        |
| ----------------- | ---------------------- | ------------------- |
| Consultation Mode | Physical Visit         | Online Consultation |
| Accessibility     | Limited by location    | Accessible remotely |
| Cost              | Higher                 | Lower               |
| Time Consumption  | More                   | Less                |
| Technology Usage  | Minimal                | High                |

---

### Prompt 2:

“Provide a table listing five healthcare technologies, their purpose, and one application.”

### AI Response:

| Technology                | Purpose             | Application          |
| ------------------------- | ------------------- | -------------------- |
| AI Diagnostics            | Disease Detection   | Cancer Screening     |
| Wearable Devices          | Health Monitoring   | Fitness Tracking     |
| Telemedicine              | Remote Consultation | Video Appointments   |
| Electronic Health Records | Data Management     | Hospital Systems     |
| Robotic Surgery           | Surgical Assistance | Precision Operations |

---

## Evaluation

| Criteria           | Observation |
| ------------------ | ----------- |
| Readability        | Very High   |
| Organization       | Excellent   |
| Data Comparison    | Effective   |
| User Understanding | Improved    |

# 3. Preceding Question Prompting

## Definition

Preceding Question Prompting involves asking a leading question before the main question to provide context and improve reasoning.

## Examples

### Prompt 1:

“Why is digital healthcare becoming important? Explain how telemedicine improves healthcare accessibility.”

### AI Response:

Digital healthcare is becoming important because it improves healthcare access, reduces costs, and enables faster communication between patients and doctors. Telemedicine allows patients to consult doctors remotely using internet-based technologies, especially benefiting people in rural and remote areas.

---

### Prompt 2:

“How do vaccines protect people from diseases? Describe the process of immunization in simple terms.”

### AI Response:

Vaccines protect people by training the immune system to recognize and fight harmful pathogens. During immunization, weakened or inactive forms of microorganisms are introduced into the body, helping the immune system build defense mechanisms without causing serious illness.

---

## Evaluation

| Criteria          | Observation |
| ----------------- | ----------- |
| Context Awareness | High        |
| Reasoning Quality | Very High   |
| Explanation Depth | Detailed    |
| Logical Flow      | Excellent   |

---

# 4. Missing Word Prompting

## Definition

Missing Word Prompting provides incomplete sentences where the AI fills in the missing information based on context.

## Examples

### Prompt 1:

“The human heart pumps ____ throughout the body.”

### AI Response:

“blood”

---

### Prompt 2:

“In telemedicine, doctors communicate with patients using ____ technologies.”

### AI Response:

“digital”

---

## Evaluation

| Criteria        | Observation |
| --------------- | ----------- |
| Simplicity      | Very High   |
| Accuracy        | High        |
| Educational Use | Excellent   |
| Response Length | Short       |

# Advantages of Prompt Engineering

1. Improves AI response quality.
2. Enhances contextual understanding.
3. Reduces ambiguity in outputs.
4. Helps generate structured responses.
5. Improves reasoning and explanation capability.
6. Enables better interaction with AI systems.

---

# Limitations

1. Poorly written prompts may produce inaccurate responses.
2. Complex prompts may confuse the model.
3. Different AI systems may interpret prompts differently.
4. Responses may still contain factual errors.

# OUTPUT

The AI system successfully generated responses for all four prompting techniques. Each prompting style produced different levels of detail, structure, reasoning, and clarity depending on the nature of the prompt.

---

# RESULT

The various types of prompts were executed successfully and the generated responses demonstrated that prompt structure significantly affects the quality and effectiveness of AI-generated outputs. Among the techniques tested, Tabular Prompting and Preceding Question Prompting produced the most detailed and organized responses, while Straightforward and Missing Word Prompting were effective for simple and direct interactions.

# Conclusion

The experiment demonstrated the importance of Prompt Engineering in improving interactions with Generative AI systems. Different prompting techniques are suitable for different applications depending on the desired output format and complexity. Properly structured prompts significantly improve response quality, reasoning capability, readability, and contextual accuracy in AI-generated content.
